🧪 Test-Barcodes zum Abscannen

Diese Codes stammen aus echten Artikeln deines Stamms. Scanne sie mit dem Handy direkt vom Bildschirm, um den ganzen Ablauf (Scan → Erkennung) zu testen — praktisch, solange du keine gedruckten Etiketten zur Hand hast. Der letzte Code ist absichtlich nicht im Stamm, damit du auch die „Unbekannt"-Meldung siehst.

Verlauf

    So testest du Phase 1:
    1. Ohne Barcode: Tippe oben eine Artikelnummer ein (z. B. 100742 oder GES1758509) → „Prüfen". Die App sollte den Artikel erkennen.
    2. Mit Kamera: Öffne „Test-Barcodes zum Abscannen", starte den Scan und halte das Handy auf einen der Codes am Bildschirm.
    3. Probiere einen unbekannten Code (z. B. 99999999) → es muss „Unbekannter Code" erscheinen.

    Hinweis: Die EAN auf einer Herstellerverpackung wird i. d. R. nicht gefunden — im Artikelstamm gibt es (noch) keine EAN-Spalte. Der zuverlässige Weg ist später das eigene MV-Etikett.
